Description:

This is the first set of modules inteded for use by the TPP-GIS Team at the Texas Department of Transportation. All tools are considered in revision and further versions will be forthcoming. This toolset is intended as a collaborative effort.

To intall:

Unzip the folder containing the module, setup.py, and MyModules.py

Open a command line and type:
1> cd [folder path]
2> python setup.py install

Windows executable is also available.
